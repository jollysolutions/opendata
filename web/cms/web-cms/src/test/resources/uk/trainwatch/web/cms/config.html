Configuration

Overview

The configuration is stored under the /etc/weather directory and consists of a series of files containing all of the required configuration for the main daemon.

The name's of each file is not important - just the presence of one or more files in that directory.

Each file is in the ini format, consisting of sections labeled [name] where name is the section name listed below.

Sections

Section name Description
annotation Defines the text, font & colours used when annotating images
camera Configures the camera
clouds Configures the cloud sensor works
logging Configures logging
observatory Defines the weather stations location
webserver Configures the embedded webserver
